# ambient-player

## app 컨셉

고양이에게 이어폰을 꽂아봐~

## 구현할 기능

- [ ] 귀여운 고양이 애니메이션 구현
  - [ ] 고양이 눈동자 마우스 따라가기
  - [ ] 고양이 귀 쫑긋 루프 애니메이션
- [ ] 이어폰 드래그 앤 드롭 시 음악 실행
- [ ] 사운드 플레이어 (빗소리, 불, 사람소리, 숲속) 
    - [ ] 각 사운드 소리 조절 기능
- [ ] 현재 시간 표시 
- [ ] (advance) 실시간 사용자 프로필과 메세지 노출
